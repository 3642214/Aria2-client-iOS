# Aria2-client-iOS

最近在小米路由器上装了Aria2，很好用，但搜了圈iOS的客户端，发现下架了，github也没找到好用的，只好自己写个。

现支持功能

 1. 添加多个jsonrpc服务器，暂不支持secret；
 2. 添加下载链接（addUri），暂不支持addTorrent、addMetalink；
 3. 下载支持暂停pause，恢复unpause，删除remove，删除记录removeDownloadResult；
